🛠️ ТЕХНОЛОГИЧЕСКИЙ СТЕК:

Backend:
  • Python 3.8+
  • FastAPI 0.104.1 (современный веб-фреймворк)

  • React Router 6.20 (маршрутизация)
  • Axios 1.6 (HTTP клиент)
  • CSS3 (стили)

📞 РЕШЕНИЕ ПРОБЛЕМ:

Проблема → Решение:

Порт занят          → Используйте другой порт (--port 8001)
CORS ошибка         → Проверьте что backend запущен
npm не найден       → Переустановите Node.js
Python не найден    → Переустановите Python
БД работает странно → Удалите medical_center.db и перезагрузите

Смотрите WINDOWS_GUIDE.md для подробных решений!

📈 РАСШИРЕНИЕ ПРОЕКТА:

Легко добавить новые функции:

1. Новая модель:
   - Создать file.py в app/models/
   - Добавить в app/models/__init__.py

2. Новые API endpoints:
   - Создать router в app/routers/
   - Добавить include_router в main.py

3. Новая страница:
   - Создать Page.jsx в src/pages/
   - Добавить Route в App.jsx

4. Новый компонент:
   - Создать Component.jsx в src/components/
   - Импортировать где нужно
